# Clover | Full-Stack E-Commerce

**Clover** is a PERN stack e-commerce application with a public storefront and an admin dashboard.

## Main Features

- Public product catalog with category-based browsing
- Product images and cart flow for shopping
- Admin dashboard for managing products, orders, customers, and messages
- Authentication for customer accounts
- PostgreSQL-backed data storage
- Responsive UI built with modern React and Vite

## Tech Stack

- Frontend: React 19, Vite, Lucide React, Recharts
- Backend: Node.js, Express, Multer, JWT, bcrypt
- Database: PostgreSQL
- Tooling: npm, ESLint
- AI : Gemini, Claude, Github Copilot
